# 박은진 포트폴리오

Java 풀스택 개발자 박은진의 포트폴리오 사이트입니다.

- 정적 HTML/CSS/JS (빌드 도구 없음)
- 다크모드, 반응형, 스크롤 애니메이션 지원

## 로컬 실행

`index.html`을 브라우저로 바로 열거나, 정적 서버로 띄워서 확인할 수 있습니다.

## 배포

GitHub Pages 기준: 저장소의 `Settings > Pages`에서 배포 브랜치를 `main`, 폴더를 `/ (root)`로 설정하면
`https://<github-id>.github.io/<repo-name>/` 주소로 배포됩니다.
